The Problem

The look-alike
site problem.

Two client sites both running 192.168.1.x behind a .1 gateway are indistinguishable to automatic network detection — there is no signal to tell them apart, so most scanners merge them into one device list.

NetScanPro fixes this by letting you draw the boundary yourself: create a named network for each site, pin your scan to it, and the records stay separate no matter how identical the sites look.
